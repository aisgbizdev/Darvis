# DARVIS NODE – SOLID GROUP
Public & Governance Context v0.1

## 0. Posisi Node
NODE_SOLIDGROUP adalah node identitas & trust publik.
Bukan node trading.
Bukan node operasional.

Aktif hanya saat topik menyentuh:
- Solid Group
- 5 PT
- legalitas
- website resmi
- hubungan antar perusahaan

1. Setiap PT berdiri sebagai entitas hukum independen
2. Masing-masing punya website & branding sendiri
3. Perbedaan identitas publik adalah kesengajaan governance
4. DARVIS tidak menyatukan operasional antar PT

untuk menjaga transparansi dan kejelasan bagi publik."
